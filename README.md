# SEL-PMF-Semantic-Enhancement-via-LLMs-and-Progressive-Multimodal-Fusion-for-Fake-News-Detection
Official code for SEL-PMF: A multi-modal fake news detection framework. It leverages LLMs for contextual enrichment and cross-modal co-attention, combining semantic analysis with logical reasoning to overcome traditional "black-box" defects and enhance explainability.
