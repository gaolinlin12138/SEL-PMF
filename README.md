# SEL-PMF
Code repository for "SEL-PMF: Semantic Enhancement via LLMs and Progressive Multimodal Fusion for Fake News Detection". It integrates logical LLM priors with DFT-based physical evidence to improve detection accuracy.
# Dataset

(1) Fakeddit Dataset: Fakeddit is an innovative multimodal dataset, with data sourced from the social news and discussion website Reddit. Fakeddit includes over 500,000 multimodal data pieces from 22 different Reddit subforums. Among them, news content from 8 Reddit subforums is real and accurate, while fake news samples are verified through manual labeling, social media user feedback, and third-party fact-checking tools. The new dataset link:https://github.com/entitize/Fakeddit


(2) Twitter Dataset: The Twitter dataset is built based on user-generated content on social media platforms, focusing on tweet samples related to specific social events. The dataset consists of 17 events, containing tweet text, accompanying images, and other social context information. The veracity of each tweet in the dataset is manually labeled or crowdsource-audited, classified using real/fake labels. The new dataset link:http://www.multimediaeval.org/mediaeval2015/verifyingmultimediause/


(3) Weibo Dataset: The Weibo dataset consists of information from the Weibo social platform. Weibo is a Chinese social media platform with a huge user base, and content posted covers various fields like news, entertainment, politics, and society. Real news samples come from China’s famous Xinhua News Agency, while fake news samples are verified through the official Weibo fake news refutation system. The new dataset link:https://docs.google.com/forms/d/e/1FAIpQLSdI0H90LyFs3ZZdH0hIP45FUl_kgsIqRfsfING1WJgYJmbeKw/viewform?pli=1
